# 介绍
``` html
<!-- 默认：宽300 高150 -->
<canvas>
    不支持canvas的浏览器 
</canvas>
```
## 路径
`beginPath`:开始绘制路径  
`moveTo`:移动到绘制的新目标点  
`lineTo`: 新的目标点  
`closePath`:结束绘制路径，闭合，可以作为起止点闭合的连接  
  
`ctx.rect(100,100,100,100)`:绘制一个方块路径
